### Instalação

1. Clone o repo
   ```sh
   git clone -b daniel-vega-lopes https://github.com/Agev-Projects/desafio-sharenergy-2021-22.git
   ```
2. Instale os pacotes
   ```sh
   npm install
   ```
3. Inicie o cliente
   ```sh
   npm start
   ```
